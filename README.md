# projekt1
Kursmanagement Interface
INF eLearn - 
Dieses Projekt ist ein statischer Prototyp für eine E-Learning-Plattform, die speziell auf die Bedürfnisse von Studierenden zugeschnitten ist. Der Fokus liegt auf einer klaren Informationsarchitektur und einem benutzerfreundlichen Layout für akademische Inhalte.

Features
Vollständiges Dashboard-Layout: Eine strukturierte Benutzeroberfläche mit Header, Sidebar und Hauptinhaltsbereich.

Akademische Sidebar: Schneller Zugriff auf Kursmaterialien (Folien, Übungen), FAQ, Datenschutz und Klausur-Informationen.

Stundenplan-Integration: Ein übersichtlicher Vorlesungs- und Übungsplan, der mittels strukturierter HTML-Tabellen umgesetzt wurde.

Authentifizierungs-UI: Integriertes Login-Feld im Header-Bereich.

Navigation: Unterstützung für Kurs-spezifische Untermenüs (Bewertung, Teilnehmer, Kursinhalte).

Technologien
Sprachen: HTML5, CSS3

Konzept: Semantic HTML & Layout-Design

Tooling:  GitHub

Projektstruktur
index.html - Die gesamte Struktur der Plattform inklusive der komplexen Tabellen-Struktur.

style.css - Das Styling für die Navigation, die Sidebar und das responsive Tabellen-Layout.

Fokus dieses Projekts
In diesem Projekt lag mein Schwerpunkt darauf, komplexe Datenstrukturen (wie einen wöchentlichen Übungsplan) visuell ansprechend und logisch in HTML umzusetzen. Es dient als Beweis für meine Fähigkeiten im Bereich UI-Design und CSS-Layouting, noch bevor dynamische Logik hinzugefügt wird.
